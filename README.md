# Excelencia Filmes - Sistema de Gerenciamento

Sistema de gerenciamento de projetos audiovisuais com quadro Kanban e Inteligência Artificial.

## Repositório
[https://github.com/ExcelenciaFilmes/excelenciafilmessys.git](https://github.com/ExcelenciaFilmes/excelenciafilmessys.git)

## Tecnologias

- React + Vite
- TypeScript
- TailwindCSS
- Supabase (Banco de Dados + Auth)
- Google Gemini AI (Geração de Roteiros e Checklists)
- @hello-pangea/dnd (Drag and Drop)

## Como rodar localmente

1. `npm install`
2. `npm run dev`

## Deploy

Este projeto está configurado para deploy automático na Vercel.
